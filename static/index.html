Synthy 3

Synthy 3 is a collaborative note sender.

(This is a very early version so will no doubt have bugs. It doesn't work on Safari yet. Please add bugs, ideas, pull request improvements to github.com/filipnest/synthy3). Open source.

Instructions

Synthy 3 has three parts:

  • A button for triggering notes to send to the world (hold down for longer notes)
  • A dial for changing pitch*
  • A binary number channel picker for choosing which channel to send / listen to.

The channel picker takes a series of long or short presses to mean 0 or 1 in a binary sequence. For example: short press + long press is 2. The channel is the current channel you are both sending notes to, and receiving notes from. To change channel, press in a series of short and long presses and then wait a few seconds.

* - Pitch is based on the 0 - 127 MIDI note frequencies (MIDI out support is coming).
